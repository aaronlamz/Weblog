要交易港美股，你需要一个**海外券商账户**。这一篇我们对比目前散户最常用的两家——富途和盈立证券，讲清楚怎么开户、怎么出入金。

> ⚠️ **重要：富途已停止接受中国大陆新客户**
>
> 受中国证监会监管要求影响，**富途和老虎证券自 2023 年起已停止为中国大陆居民开立新账户**。具体表现：
> - 新的大陆身份证用户**无法注册新账户**
> - 大陆已有客户**保留账户、可继续交易**，但部分功能（如新入金）有所限制
> - 如果你是大陆居民、还没有富途账户，**只能选择盈立或其他仍接受大陆客户的券商**
>
> 如果你已经有富途账号（在限制前开通的），可以继续使用。下面的对比依然有参考价值。

## 为什么不选传统券商

很多人会问：为什么不选老牌的盈透证券（IB）、嘉信、第一证券？

**答案**：对中文用户来说，这些券商：
- 界面英文为主，对新手不友好
- 客服基本不懂中文
- 入金渠道复杂（要电汇）
- 没有中文研究报告和资讯

而**富途和盈立**专为中港用户设计：
- 中文界面 + 中文客服
- 支持港币、美金账户
- 有手机 App，操作直观
- 提供中文的财报、研究报告
- 入金方便（支持 FPS 转数快、电汇）

## 富途证券 vs 盈立证券

| 维度 | 富途证券 | 盈立证券 |
|---|---|---|
| **公司背景** | 富途控股（NASDAQ: FUTU） | uSmart Securities |
| **监管** | 香港 SFC、美国 SEC、新加坡 MAS | 香港 SFC |
| **可交易市场** | 港、美、A、新 | 港、美 |
| **港股佣金** | 0.03%（最低 3 港币） | 0%（永久免佣） |
| **美股佣金** | 0.0049 美元/股 | 0% 永久免佣 |
| **平台费** | 港股有，美股无 | 港股有，美股无 |
| **App 体验** | 富途牛牛，功能强大 | uSmart，简洁 |
| **数据** | 港股 Lv1 免费、Lv2 付费 | 港股 Lv2 免费 |
| **打新** | 强大，有融资额度 | 较好 |
| **存款利息** | 美元闲置资金有利息 | 美元闲置资金有利息 |
| **客服** | 中文，回应快 | 中文，回应快 |
| **API 量化** | 强（免费 OpenAPI） | 较弱 |

### 我的建议

**如果你是大陆居民、目前还没有任何账户**：

- 首选 **盈立证券**（仍接受大陆客户开户）
- 富途已不能新开了，不用考虑

**如果你已经有富途账号**（限制前开通的）：

- **只买美股 + 偶尔港股** → 用盈立（永久免佣）
- **想做量化、需要 API** → 用富途（OpenAPI 是免费的）
- **打新股很积极** → 用富途（融资打新额度高）
- **两个都用** → 是最常见的选择，分散风险

**如果你是港澳台 / 海外身份**：

- 两家都可以正常开户，按需求选择

我自己两个都开了（在限制前开通的），**富途用来量化和打新，盈立用来美股长持**。

### 大陆客户的其他选择

如果你是大陆居民、又错过了富途的开户窗口，除了盈立，还有：

- **老虎证券** — 同样停止接受新大陆客户
- **盈透证券（IB）** — 全英文，门槛高，仍可开户
- **嘉信、第一证券** — 美国本土券商，部分仍接受大陆客户但难度大
- **微牛**（Webull） — 美国上市券商，曾有大陆开户通道

> 💡 **现实建议**
> 对大陆新手来说，**盈立证券是目前最务实的选择**——中文友好、入金方便、永久免佣。本系列后续涉及具体操作时，默认以盈立为例。

## 开户流程

港美股券商开户都是**线上完成**，整个过程 30 分钟左右。

### 准备材料

```
□ 身份证（中国大陆）或港澳通行证
□ 地址证明（3 个月内的水电煤账单 / 银行账单 / 信用卡账单）
□ 一张内地银行卡（用于入金）
□ 手机号
□ 邮箱
```

### 步骤

以富途为例（盈立流程几乎一样）：

1. **下载 App**（富途牛牛 / uSmart）
2. **注册账号** — 用手机号注册
3. **选择开户类型** — 个人账户
4. **填写基本信息** — 姓名、身份证、地址
5. **上传证件照片** — 身份证正反面、地址证明
6. **风险评估问卷** — 评估你的投资经验和风险承受能力
7. **签署协议** — 电子签名
8. **人脸识别** — 视频认证
9. **等待审核** — 通常 1-3 个工作日

审核通过后，你会收到通知，账户开通成功。

## 入金（最关键的一步）

### 入金渠道

| 方式 | 速度 | 手续费 | 额度限制 |
|---|---|---|---|
| **FPS 转数快**（港币） | 几分钟 | 免费 | 较高 |
| **eDDA 自动直通** | T+1 | 免费 | 单日 1 万港币左右 |
| **电汇** | T+1 - T+3 | 100-300 港币 | 高 |
| **银证通** | 实时 | 免费 | 看券商支持 |

### 内地资金怎么过去

这是最让新手头疼的问题。中国大陆有 **5 万美元年度购汇额度**，超出需要其他途径。

**主流方法**：

1. **信用卡 / 储蓄卡换汇 → 香港银行卡 → 券商**
   - 需要先开一张香港银行卡（汇丰、中银香港、众安银行等）
   - 内地银行 → 香港银行 → 券商
   - 受 5 万美元年度额度限制

2. **数字银行**
   - 使用众安银行、ZA Bank、Mox 等虚拟银行的香港账户
   - 跨境转账方便

3. **直接电汇**
   - 内地银行 → 券商指定收款行
   - 备注用途（买卖证券需要）
   - 注意各银行对个人购汇用途的审核

> ⚠️ **合规提醒**
> 中国大陆居民购汇用于"境外证券投资"在严格意义上属于**资本项下**，不在 5 万美元用途范围内。实际操作中很多人在做，但要了解监管风险。具体合规问题请咨询专业人士。

## 出金

出金路径基本是入金的逆向：

- 券商 → 香港银行账户 → 内地银行
- 或：券商 → 直接电汇到内地

**出金通常需要 1-3 个工作日**，部分会被银行风控审核（特别是大额）。

> 💡 **建议**：开户初期就走通"小额入金 + 小额出金"流程，确认资金能进出，再加大投入。

## 美股 vs 港币账户

券商账户里通常有两个独立子账户：
- **美元账户**：用来买美股
- **港币账户**：用来买港股

**重要**：买港股不能直接用美元，需要先换汇到港币，反之亦然。换汇有汇率差，每次 0.1% 左右。

### 汇率小知识

- USD/HKD 汇率长期固定在 7.75-7.85（港币联系汇率制度）
- USD/CNY 汇率波动较大
- 富途和盈立都支持**自动换汇**或**手动换汇**

## 第一次入金多少合适

新手最容易犯的错：**一上来 ALL IN**。

建议：

```
第一次入金：1000-3000 美元 / 1 万港币
```

这点钱用来：
- 熟悉下单流程
- 体验交易感受
- 试错（必然会犯错）

**等你能稳定盈利或者至少不犯低级错误后，再加大投入**。

## 开户后的第一个动作

不要急着买股票。先做这些：

1. **设置交易密码**
2. **开通行情订阅**（港股 Lv1 免费，Lv2 看券商政策）
3. **熟悉 App 界面** — 行情、自选股、资金账户、交易功能
4. **看几篇研报** — 富途和盈立都有
5. **设置 K 线图样式**（个人喜好）
6. **添加一只股票到自选** — 比如苹果（AAPL）

## 小结

完成这一篇你应该已经：

- ✅ 知道富途和盈立的区别
- ✅ 完成开户流程
- ✅ 完成第一次小额入金
- ✅ 熟悉了 App 基本功能
