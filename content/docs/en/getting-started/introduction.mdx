## Welcome

This is a modern blog system built with Next.js, featuring MDX content, internationalization, dark mode, and more.

## Tech Stack

- **Next.js 15** — Full-stack React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling solution
- **MDX** — Markdown + JSX content format

## Key Features

### Blog System

Write articles in MDX format with automatic table of contents, syntax highlighting, and reading time estimation.

### Internationalization

Built-in Chinese and English support, easily extendable to more languages.

### Documentation Center

What you're reading right now — organized by categories with sidebar navigation.
