## 欢迎

这是一个基于 Next.js 构建的现代博客系统，支持 MDX 内容、国际化、暗色模式等特性。

## 技术栈

- **Next.js 15** — React 全栈框架
- **TypeScript** — 类型安全
- **Tailwind CSS** — 样式方案
- **MDX** — Markdown + JSX 内容格式

## 主要功能

### 博客系统

支持 MDX 格式的文章编写，自动生成目录，代码高亮，阅读时间估算。

### 国际化

内置中英文双语支持，可以轻松扩展更多语言。

### 文档中心

你正在阅读的就是文档系统，支持分类组织、侧边栏导航。
